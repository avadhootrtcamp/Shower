Introduction

rtMedia

  • Free audio/video conversion
  • WordPress Integration - Display media on WordPress author pages
  • BuddyPress Integration - Find a new media tab under BuddyPress Profiles & Groups
  • BuddyPress Activity Stream - Attach media to activity status updates just like Facebook

Installation

Easiest possible method!

Get it started!

  • In your dashboard: Go to Plugins > Add New > Search and search for rtMedia
  • Or download the plugin from the repository. Unzip it and upload it to wp-content/plugins/ directory of your WordPress installation
  • Activate the plugin

Features

rtMedia Features

  • Responsive - Album Slideshow (Lightbox), video player, uploads work on mobiles & tablets. Support for swipe guestures
  • Albums - Organise media into manageable collections. BuddyPress Group albums support collaboration
  • Privacy - Control who can see media files, albums and BuddyPress activities
  • Templating system - Completely customise rtMedia by modifying the template files

And some more amazing features!

  • Featured Media - This can be used for facebook-style cover photo on profiles
  • rtMedia Uploader - Use [rtmedia_uploader] shortcode or template tag, to show drag-n-drop uploader in any WordPress area
  • rtMedia Gallery - Display media gallery anywhere on your site using [rtmedia_gallery] shortcode or template tag.
